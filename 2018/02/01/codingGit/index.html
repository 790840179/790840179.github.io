1.网络下载至新文件夹

git clone https://git.coding.net/narisolar/apogee.git

git branch zhouguang 新建本地分支zhouguang

2.网络下载项目并与本地项目合并

git checkout master 切换至本地master

git pull 将网络master拉至本地master

git checkout zhouguang 切换至本地zhouguang

方案一：git push origin zhouguang:zhouguang

网络zhouguang没有建立，现在建立并上传本地zhouguang至网络zhouguang

方案二：

//git pull

git branch –set-upstream-to=origin/zhouguang zhouguang 本地zhouguang与远程zhouguang联系

git pull 将网络zhouguang拉至本地zhouguang

git merge master 将本地master(已经将网络master拉下来的)与zhouguang合并

3.上传本地项目至网络分支zhouguang

git status 上传

git add .

git commit -m “代码备注”

git push origin zhouguang //origin是服务器zhouguang是分支

4.合并到网络分支master

git checkout master

git merge zhouguang

git push
